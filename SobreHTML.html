Regras Básicas do HTML ☑

As tags aparecem entre os sinais de <>

    Como por exemplo:
    <b> <p> <th> <span> <script>

As tags são utilizadas em pares em sua maioria

    <b> </b>
    <p> </p>
    <h1> <h1>
    <span> </span>
    <div> </div>
    <tbody> </tbody>

Algumas tags são fechadas nelas mesmas

<img src="" alt="">

Estrutura Básica do HTML

<html> ⇒ Página
<head> ⇒ Cabeçalho da Página
<title> ⇒ Título do documento
</head> ⇒ Fim do Cabeçalho
<body> ⇒ Início do Corpo (Aqui inicia os códigos)
<link rel = "stylesheet" type="text/css" href="styles.css">
↳ Aqui chama um link externo de CSS
</body> ⇒ Final do Corpo
</html> ⇒ Final do Documento HTML

Estilo Itálico

Estilo Riscado
